# cleanheap

To use:

```bash
npx cleanheap <input-file-path>
```

Optionally:

```bash
npx cleanheap <input-file-path> <output-file-path>
```

## Contributing

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run src/index.ts
```

To compile:

```bash
bun build src/index.ts --outfile bin/cleanheap
```
